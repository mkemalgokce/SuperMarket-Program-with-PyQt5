## SuperMarket Product Program with PyQt5
Market product program is the Python desktop application that developed for supermarket management.  
My first try with PyQt5. 

## Requires
- Python 3 
- PyQt5 

## Installation
- pip install PyQt5 

## Run
- python login.py.   
- Default id = admin.  
- Default pass = 1234.  
- You can change these in constant.py.
